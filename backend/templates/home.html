Movie API

Simple movie API using OMDB data. Here's what you can do:

Get Movie Details

GET /api/movie/?title=The Matrix

Get info about a movie by its title

Get TV Episode Info

GET /api/episode/?series_title=Breaking Bad&season=1&episode_number=1

Get details about a specific TV episode

Find Movies by Genre

GET /api/movies/genre/?genre=Action

Get top 15 movies in a genre, sorted by rating

Note: All responses are in JSON format
